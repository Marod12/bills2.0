# bills2.0

Este projeto em python permite que você guarde e visualize dados de forma criptografada. Versão melhorada do [Bills1.0](https://github.com/Marod12/bills1.0), ultilizando sqlite3 no lugar de arquivo `.txt`

## Como executar

`
 python main.py
`
